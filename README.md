## ğŸ“Š Project Description

This Power BI project presents a comprehensive **financial performance dashboard** based on a sample Excel dataset. It analyzes **Gross Sales**, **COGS (Cost of Goods Sold)**, and **Profit** data. The project showcases data transformation using **Power Query**, **data modeling**, and **DAX measures** to deliver interactive, multi-perspective business insights.

The dashboard includes three distinct performance views:
- **Gross Sales View**
- **COGS View**
- **Profit View**

Each view allows analysis across multiple dimensions: **monthly**, **country-wise**, **segment-wise**, and **discount band-wise**.

## ğŸ“Œ Key Features

### 1. **Gross Sales View**
- **Line Chart**: Monthly gross sales performance
- **Stacked Column Chart**: Country-wise gross sales
- **Ribbon Chart**: Discount band-wise gross sales
- **Waterfall Chart**: Segment-wise gross sales
- **Bookmarks**: Enable switching between views

### 2. **COGS View**
- **Stacked Bar Chart**: Monthly COGS performance
- **100% Stacked Bar Chart**: Country-wise COGS breakdown
- **100% Stacked Area Chart**: Discount band-wise COGS
- **Donut Chart**: Segment-wise COGS performance

### 3. **Profit View**
- **Matrix**: Monthly profit performance
- **Map Visual**: Country-wise profit analysis
- **Multi-row Card**: Discount band-wise profit metrics
- **Pie Chart**: Segment-wise profit distribution
